招聘职位

  • JAVA工程师

    岗位职责：

    - 参与企业管理业务系统设计、开发与实现；

    - 结合需求设计实现安全、稳定、高伸缩性、高性能、易维护、易用性好的业务系统；

    - 跟踪并研究最新技术并能应用于产品。

    任职资格：

    - 3 年以上 Java 和 Web 开发经验；

    - 具备面向对象的编程思想，熟悉设计模式；

    - 具备扎实的 Web开发基础，能够熟练使用 Jsp 、Servlet 、Spring 、Hibernate，深刻理解 HTML, JavaScript, CSS；

    - 具备扎实的数据库基础，Oracle优先；

    - 熟练使用 SVN版本管理工具，并对软件配置管理的概念和方法有所了解；

    - 富有进取精神及团队合作精神，良好的沟通能力,能够承受压力；

    - 责任心强，善于合作沟通。

    岗位职责：

    任职资格：

    - 熟练掌握.Net Framework常用类库的使用方法；

    - 拥有WPF或Silverlight的项目开发经验；

    - 熟悉数据库相关知识；

    - 有泛型、委托、反射、Http请求接口及多线程开发经验者优先考虑；
